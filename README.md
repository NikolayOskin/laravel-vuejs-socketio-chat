# Laravel Vue.js Socket.io Chat (+ Redis)
Simple Laravel Vue.js Socket.io Chat (+ Redis)

### Server requirements:
- LAMP or LEMP stack
- Node.js
- Redis
- Composer

### How to use:
1. Clone or download
2. Change dir `cd laravel-vuejs-socketio-chat`
3. Run `composer install`
4. Run `npm install` (node.js should be installed)
